!!!Contact Management System (Android App)!!!

1. Project Overview:-A simple Android application to manage contacts efficiently.The app allows users to add, view, edit, and delete contact information
   with secure authentication and real-time data storage using Firebase.

2. Features :
   - User login using Firebase Authentication
   - Add new contacts (Name, Phone, Email)
   - View contacts in RecyclerView
   - Edit existing contacts
   - Delete contacts with confirmation
   - Real-time database updates

3. Technologies Used :
    - Frontend: XML
    - Backend: Java
    - IDE: Android Studio
    - Firebase Authentication
    - Firebase Realtime Database
    - RecyclerView
